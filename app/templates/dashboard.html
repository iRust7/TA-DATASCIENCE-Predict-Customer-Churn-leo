Machine Learning Deployment Workflow

Veltrix is perfectly configured to deploy Machine Learning, Data Science, and AI models. Follow this simple checklist to deploy your model:

  • Model Training: Save your trained model (e.g., joblib.dump(model, 'model.pkl')).
  • Model Asset: Place your .pkl or .onnx file inside app/static/ or a custom assets folder.
  • Endpoint Setup: Write custom routes in app/routes/main.py to load the model and predict inputs.
  • API Testing: Test predictions via POST request using tools like curl or Postman.
  • Prediction Logs: Log prediction inputs and results directly into the database using our SQLAlchemy user schema model template.
